# 🚀 Júpiterlang (jupiterlang)

**Júpiterlang** is a **safe programming language** that transpiles to **modern C++** with **smart pointers** for memory safety.

---

## 📦 How to Use:
```bash
g++ -std=c++23 src/jupitercc.cpp -o jupitercc
./jupitercc examples/hello.jupiter
```

This will generate a file called `out.cpp` containing the transpiled C++ code.


---

# 📄 Official Extension:

`.jupiter`
`.jup`

---

## ✨ Key Features:

**Safe by design: Smart pointers** (like `std::shared_ptr, std::unique_ptr`) for automatic memory management.

Transpiles to C++: Easy integration with C++ projects.

Minimal & Modern Syntax: Inspired by C++ and Rust.

Future plan: full compiler bootstrap in Júpiterlang itself.



---

## 🔥 Example:
```jupiter
hello.jupiter

let foo = 42;

print(foo)
```
---

## Output C++ (generated by jupitercc):
```cpp
// C++ code generated by Júpiterlang
#include <iostream>

int main() {
    auto foo = 42; // example variable
    std::cout << foo << std::endl;
    return 0;
}
```
